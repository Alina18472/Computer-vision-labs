# Лабораторные работы по обработке изображений

## Лабораторная работа №1: Базовые параметры изображений

В рамках этой работы было разработано desktop-приложение с графическим интерфейсом для анализа и редактирования изображений.

### Реализованный функционал

#### Загрузка и отображение
- Загрузка изображений с локального диска
- Отображение загруженного изображения в интерфейсе

#### Анализ изображения
- **Основная информация:**
  - Размер файла
  - Разрешение (ширина × высота)
  - Глубина цвета
  - Формат файла
  - Цветовая модель
- **EXIF-данные** (не менее 5 параметров):
  - Дата съемки
  - Модель камеры
  - Параметры экспозиции (выдержка, диафрагма, ISO)
  - Фокусное расстояние
  - Производитель камеры
- **Дополнительная информация:**
  - Соотношение сторон
  - Общее количество пикселей

####  Редактирование изображений
- Преобразование цветного изображения в градации серого
- Регулировка параметров с помощью ползунков:
  - **Яркость**
  - **Насыщенность** 
  - **Контрастность**

#### истограммы
- Построение общей гистограммы изображения
- Отображение гистограмм по каналам R, G, B
- Сравнение гистограмм "до" и "после" коррекции
- Визуализация изменений после регулировки яркости, насыщенности и контрастности

####  Коррекция ЧБ изображений
- Линейная коррекция 
- Нелинейная коррекция



## Лабораторная работа №2: Морфологические преобразования 

Вторая лабораторная работа посвящена реализации морфологических операций и различных фильтров для обработки изображений.

### Реализованный функционал

#### Загрузка и отображение
- Загрузка исходного изображения
- Отображение оригинального и обработанных изображений

#### Морфологические операции
Реализованы основные морфологические операции с возможностью задания структурного элемента через матрицу:

- **Эрозия** - уменьшает размеры объектов в изображении. Все объекты, меньшие чем структурный элемент, стираются, что приводит к удалению мелких шумов.
- **Дилатация** - увеличивает размеры объектов, "расширяя" их за счет добавления пикселей к границам. Помогает соединять разрозненные части и заполнять небольшие дыры.
- **Открытие** (эрозия → дилатация) - используется для удаления мелких объектов или шумов. Эрозия удаляет мелкие детали, а дилатация восстанавливает форму больших объектов.
- **Закрытие** (дилатация → эрозия) - помогает закрыть маленькие дыры внутри объектов и соединить близко расположенные структуры. Дилатация заполняет дыры, а эрозия восстанавливает форму.
- **Градиент** (дилатация - эрозия) - используется для выделения границ объектов. Подчеркивает области, где происходит значительное изменение значений пикселей.
- **Чёрная шляпа** (оригинал - эрозия) - выделяет темные объекты на ярком фоне, усиливая мелкие темные детали.
- **Цилиндр** (оригинал - открытие) - усиливает яркие объекты на темном фоне.

#### Фильтры и эффекты
- **Повышение резкости** - усиление четкости изображения
- **Размытие в движении** - создание эффекта движения камеры
- **Эффект "Тиснение"** - придание объемности изображению
- **Медианная фильтрация** - эффективное подавление шума "соль-перец"

### Особенности реализации
- **Гибкий структурный элемент** - возможность задания произвольной матрицы для морфологических операций
- **Сравнение результатов** - одновременное отображение исходного и обработанного изображений
- **Интерактивный интерфейс** - удобное управление параметрами операций

## Лабораторная работа №3: Работа с контурами и ключевыми точками

Третья лабораторная работа посвящена разработке приложения для анализа изображений с акцентом на выделение границ, сегментацию и обнаружение ключевых точек. Приложение предоставляет удобный графический интерфейс для выполнения сложных операций компьютерного зрения.

### Реализованный функционал

#### Загрузка и отображение
- Интуитивная загрузка изображений через диалоговое окно выбора файлов
- Отображение оригинального изображения в основной области просмотра
- Функция восстановления исходного изображения для сброса примененных эффектов

#### Детектирование границ
Реализованы два метода выделения контуров объектов:

- **Детектор Canny** - использует два порога для подавления шума и выделения значимых границ. Алгоритм эффективно находит резкие изменения яркости, соответствующие контурам объектов.
- **Оператор Робертса** - применяет перекрестные операторы для вычисления разности яркости между соседними пикселями. Результат представляет собой четкие контуры в виде ярких линий на темном фоне.

####  Сегментация изображений
- **Кластеризация K-средних** - разделение изображения на заданное количество кластеров (от 2 до 20)
- **Интерактивный ввод** - пользователь может динамически изменять количество кластеров
- **Визуализация результатов** - сегментированное изображение отображается для сравнения с оригиналом
- **Управление отображением** - возможность скрыть результаты сегментации

#### Обнаружение ключевых точек
- **Алгоритм FAST** - быстрый и эффективный метод для нахождения характерных особенностей изображения
- **Визуализация** - ключевые точки отображаются в виде зеленых кружков, выделяя углы и грани объектов

#### Распознавание автомобильных номеров
- **Метод каскадов Хаара** - машинное обучение для обнаружения объектов определенного класса
- **Автоматическое выделение** - найденные номерные знаки отмечаются прямоугольными рамками
- **Готовые модели** - использование предобученных классификаторов для точного распознавания


## Лабораторная работа №4: Сопоставление изображений. Работа с видео

Четвертая лабораторная работа посвящена разработке многофункционального приложения для работы с изображениями и видео. Приложение сочетает в себе возможности поиска визуально похожих изображений и обработки видео в реальном времени.

###  Реализованный функционал

#### Сравнение и анализ изображений
- **Загрузка наборов изображений** - возможность загружать от 3 до 10 изображений одновременно
- **Поиск похожих изображений** - автоматическое определение двух наиболее похожих изображений в наборе
- **Глубокое обучение для сравнения** - использование предобученной модели ResNet18 для извлечения признаков
- **Метрика схожести** - сравнение с помощью евклидовой метрики для точного определения сходства

#### Обработка видео в реальном времени
- **Загрузка и воспроизведение видео** - поддержка различных видеоформатов с интуитивным управлением
- **Вычитание фона** - алгоритм отделения переднего плана от фона с созданием маски движущихся объектов
- **Замена фона** - динамическая замена исходного фона видео на статичное изображение
- **Адаптивное размытие** - регулируемое размытие движущихся объектов с помощью ползунка интенсивности
- **Визуализация оптического потока** - цветовое представление направления и скорости движения объектов

### Технические особенности

#### Модуль сравнения изображений
- **Архитектура ResNet18** - использование сверточной нейронной сети для извлечения семантических признаков
- **Евклидово расстояние** - точное измерение сходства между векторными представлениями изображений
- **Валидация входных данных** - проверка количества загружаемых изображений и корректности форматов

#### Модуль обработки видео
- **Алгоритм вычитания фона** - выделение движущихся объектов через анализ различий между кадрами
- **Гауссово размытие** - применение фильтра с регулируемым ядром для контроля степени размытия
- **Оптический поток** - анализ перемещения пикселей между последовательными кадрами
- **Интерактивное управление** - динамическое применение эффектов без прерывания воспроизведения
